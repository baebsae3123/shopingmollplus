# shopingmollplus 쇼핑물추가기획서

# 유저기능

## 상품 리뷰 기능

해당 상품에 별점 0/5칸 별을 남겨두고 거기안에 
짧은 텍스트로 사용자가 신뢰하며 구매할수있도록 만들어주기

## 찜 목록 랭킹 

현재 많은 관심을 받고 있는 상품을 쉽게 확인할 수 있다.
구매를 망설이는 고객에게 사회적 증거(Social Proof)를 제공한다.
인기 상품 노출을 통해 추가 구매를 유도할 수 있다.

## 다크모드 

야간 사용 시 눈의 피로를 줄일 수 있다.
사용자 취향에 맞는 환경을 제공할 수 있다.
모바일 사용자의 만족도를 높일 수 있다.

# 관리자 기능

매출 대시보드 (재고등 보일수있도록 최근매출금액)

일별/주별/월별 매출 확인
주문 건수 확인
인기 상품 확인
재고 현황 확인
품절 예정 상품 표시
